Road- Accident-Dashbord(Interactive Dashboard creation using Ms Excel)
## Project Obejective 
The main objective of this project is to convert raw accident data into meaningful insights that can help in:
1) Understanding accident trends
2) Identifying high-risk conditions
3) Supporting better road safety decisions.

## Data Set used.
- <a href="https://docs.google.com/spreadsheets/d/1HFmqGsPIqCPNsieqrFH5jGIc3wX-bByT/edit?usp=sharing&ouid=113744491659665008374&rtpof=true&sd=true">Dataset</a>

## Questions (KPIs)
1) What is the total number of casualties?
2) How many casualties are fatal, serious, and slight?
3) What is the percentage distribution of each severity level?
4) Which vehicle type has the highest number of accidents?
5) What is the total number of casualties by each vehicle type?
6) How do accidents vary month by month?
7) Which month has the highest number of casualties?
8)  Which road type has the most accidents?
9)Are accidents higher on single carriageway or dual carriageway?

## Process
1) Data Collection
Collected road accident dataset in Excel format.
2) Data Cleaning
Removed missing values, corrected errors, and formatted the data properly.
3)Data Processing
Organized data using Pivot Tables to calculate total casualties, categories, and trends.
4) Data Analysis
Analyzed data based on:
Vehicle type
Road conditions
Location (urban/rural)
Time (monthly trends)
5) Data Visualization
Created charts like bar charts, line charts, and donut charts to represent insights.
6)Dashboard Creation
Designed an interactive dashboard with KPIs and visuals for easy understanding.
